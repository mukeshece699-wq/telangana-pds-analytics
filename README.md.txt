# Telangana PDS Analytics

## Overview

This project analyzes Fair Price Shops (FPS) using clustering techniques to identify performance patterns and anomalies.

## Features

* Data integration (Transactions + Card + Location)
* Feature engineering
* PCA + KMeans + DBSCAN clustering
* Streamlit dashboard

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Output

* Clustered shop categories
* Anomaly detection
* Interactive dashboard
